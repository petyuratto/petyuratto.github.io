Vectors
  • pCS2+hSpCas9 | Ansai and Kinoshita, 2014
    [Map] (pdf, 20KB), [Sequence] (genbank, 14KB)
    This vector can be used as a template vector for Cas9 RNA using SP6 RNA polymerase.
    The hSpCas9 (human codon-optimized Streptococcus pyogenes Cas9, Cong et al, 2013) gene was cloned into the pCS2+ vector.
    RNA produced using this vector worked very well in embryos of a number of teleost fish species.
    This vector is available via Addgene (Plasmid #51815).
